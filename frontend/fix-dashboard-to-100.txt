PRODUCT CONTEXT
==================================================

This product is a dashboard generator / UI generator SaaS platform.

Core frontend flows:

1. User creates a new project.
2. User writes a prompt.
3. User chooses a theme.
4. User sees credit cost.
5. User confirms generation.
6. User sees generation lifecycle progress.
7. User previews the generated dashboard.
8. User switches device preview:
   - Desktop 1440
   - Desktop 1280
   - Tablet 768
   - Mobile 390
9. User views generated code.
10. User opens project files.
11. User views version history.
12. User refines a section.
13. User restores previous versions.
14. User manages settings:
   - Profile
   - Generation defaults
   - Workspace
   - Security
   - API keys
   - Billing / Credits
   - Danger zone

The UX quality should feel like a premium UI generator platform similar to Shayna, but do not copy Shayna visually 1:1.
Use the product quality benchmark, not direct visual cloning.
